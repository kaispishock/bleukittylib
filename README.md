# bleukittylib
C Library that allows the easy display of arrays, as well as various math functions. Named after my friend, Bleukitty.

## implementation
in order to use bluekittylib in your program, simpley download the bluekitty.c file and copy to your project directory. Include it as a local file.
Ex.
```
#include <stdio.h>
#include "bleukitty.c"
```
